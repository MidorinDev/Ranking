# Ranking  
アスレチックのランキングシステムです。  

## 使用方法（設定方法）  
1. `/ranking setup` を実行して設定GUIを開く。  
2. 設定GUIにて中間地点を入れるか入れないかを選択する。  
　　（中間地点は二つ折り(?)になってるアスレチックの場合の使用推奨  
3. 中間地点の設定ブロックとゴール地点の設定ブロックが付与されます。  
4. 各ブロックを任意の場所に設置してください。（設置した場所が各地点となります）
## 注意事項  
初期状態ではゴール地点の設定ブロック設置時にランキングがプレイヤーリスト(Tab)にて表示されます。  
配布マップなので、アスレチックスタート時にランキング表示を開始したい場合は、  
コンフィグにて `〇〇` をfalseにして、コマンドブロックの最後尾に `/ranking` と入力したコマンドブロックを設置してください。
